# ①完売確認サービス
![Game Image](.img/XXXX.png)
## ②説明
- ラーメン店の完売状況を確認できるサービスです。今から行って食べれるかどうか？の判断に使えます！

- ログイン画面、管理画面、注文画面、確認画面の3画面があります
- あらかじめ３つのタブでデプロイURLを開いておいてください。
- ログイン画面でそれぞれのパスワード　admin order user　を入力してください
-  order画面で注文を入れると在庫数が減ります
-  管理画面では残在庫から提供可能数を確認できます
-  管理画面で待ち人数も入力できます
-  確認画面では待ち人数と残り提供可能数が表示されます

## ③アプリのデプロイURL
https://ynproject.github.io/XXXXXXXXX/
- app.js内の firebaseConfigに各自の値を入れてから使ってください。

## ④アプリのログイン用IDまたはPassword（ある場合）
- ID: 今回無し
- PW: 管理画面　admin
- PW: 注文画面　order
- PW: 確認画面　user

## ⑤工夫した点・こだわった点
- 素材の組み合わせで提供できるメニューが計算されるところ
- スマホ画面をベースにPCやタブレットでも同じレイアウトにしたこと


## ⑥難しかった点・次回トライしたいこと（又は機能）
- 検証作業が今まで以上に手間がかかった、コード変更するとログイン画面に戻るので毎回ログインして画面確認するなど
